dates auxquelles PUBG Mobile KR est injouable

2021-04-03 2 min read # pubg

Voici les dates auxquelles PUBG Mobile KR est injouable dans d'autres pays?

PUBG MOBILE propose différentes versions du jeu pour différents pays et l'une des versions les plus jouées dans le monde est la version PUBG Mobile KR. La version coréenne de PUBG Mobile est spécialement conçue par Krafton Inc et cette version du jeu présente des fonctionnalités plus intéressantes que les autres versions du jeu, c'est pourquoi il s'agit de la version la plus jouée au monde. Récemment, les développeurs du jeu ont annoncé que la version coréenne du jeu ne serait plus disponible pour les utilisateurs d'autres pays. Dans cet article, nous discuterons des dates exactes à partir desquelles les utilisateurs d'autres pays ne pourront pas jouer à la version KR du jeu.

PUBG Mobile KR injouable dans d'autres pays

Voici les dates auxquelles PUBG Mobile est injouable dans d'autres pays-

  1. À partir du 01/05/2021, les utilisateurs hors de Corée ou du Japon qui se connectent avec des comptes de build KRJP ne peuvent pas régler le paiement dans la build KRJP.
  2. À partir du 01/05/2021, les utilisateurs hors de Corée ou du Japon qui se connectent avec un compte de build KRJP peuvent rencontrer des problèmes de réseau pendant la lecture.
  3. À partir du 2021/06/30, les utilisateurs hors de Corée ou du Japon qui se connectent avec un compte de build KRJP ne peuvent plus se connecter.

Ce détail est partagé par PUBG Mobile Korea sur la chaîne officielle Instagram. Les utilisateurs de l'Inde jouant à cette version du jeu ne pourront plus accéder au jeu à partir du 30 juin 2021.
