PUBG Mobile Update 2 avril 2021: le mode dirigeable arrive bientôt !!

The Game for Peace est la version époustouflante du PLAYERUNKOWN'S BATTLEGROUNDS (PUBG) qui est disponible pour PC et mobile. Selon la dernière mise à jour, Game for peace a décemment lancé la version bêta du jeu et cette mise à jour est très excitante car les joueurs l'apprécient beaucoup.

Désormais à l'occasion du 2ème anniversaire de Game for Peace le 26 mars 2021, les développeurs du jeu ont lancé un nouveau modèle dans le jeu 'Airship Mode'. Dans ce dernier mode, les joueurs peuvent profiter du jeu en utilisant le dirigeable. Les joueurs peuvent atterrir directement sur le dirigeable et piller sur le dirigeable. Il y a beaucoup de mini-jeux fournis sur le dirigeable. Dans ce mode, les joueurs peuvent également profiter de l'espace à l'aide des fusées disponibles sur la carte.

Il existe des drones disponibles en mode dirigeable et les joueurs peuvent voler très facilement. On s'attend à ce que ce mode Airship soit bientôt disponible dans la version mondiale de PUBG Mobile. Cependant, il n'y a pas de déclaration officielle donnée par PUBG Mobile concernant la sortie du mode dirigeable.
